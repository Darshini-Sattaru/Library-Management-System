# Library Management System

A beginner-friendly, command-line based application to track book inventory and availability.
## Overview
This project provides a simple way to manage a library collection through an interactive menu. It allows users to add, search for, issue, return, and delete books from the system. It was designed as a learning project to practice Python data handling and modular code structure.
## Features
 * **Add Books**: Quickly add new titles and authors to the library database.
 * **View Collection**: See a complete list of all books, including their current availability status.
 * **Search**: Find specific books by searching for keywords in the title or author name.
 * **Issue & Return**: Update the status of books as they are checked out or returned.
 * **Delete**: Remove books permanently from your records.
## Usage
 1. Ensure you have Python installed on your computer.
 2. Run the application from your terminal or command prompt:
   ```bash
   python main.py
   
   ```
 3. Follow the on-screen menu (1–7) to manage your library.
